# 医院数据管理系统 - 系统架构文档

## 1. 系统概述

医院数据管理系统是一个综合性的数据同步和管理平台，主要实现医院多源数据的采集、同步、存储和知识库管理功能。

### 1.1 核心功能

- 用户认证与权限管理
- 多源数据库连接管理
- 数据同步任务配置与调度
- 数据同步日志监控
- 科室信息映射与管理
- 基于大模型的知识库管理

## 2. 技术架构

### 2.1 整体架构

系统采用前后端分离的架构设计：

```
┌─────────────────────────────────────────────────────────┐
│                        前端层                            │
│              React 18 + TypeScript + Ant Design         │
└─────────────────────────────────────────────────────────┘
                            │
                            │ HTTP/REST API
                            ↓
┌─────────────────────────────────────────────────────────┐
│                       后端服务层                         │
│           FastAPI + Python 3.11 + Uvicorn              │
│  ┌──────────────┐  ┌──────────────┐  ┌──────────────┐  │
│  │  Auth模块    │  │  Sync模块    │  │  Knowledge  │  │
│  └──────────────┘  └──────────────┘  └──────────────┘  │
└─────────────────────────────────────────────────────────┘
                            │
        ┌───────────────────┼───────────────────┐
        ↓                   ↓                   ↓
┌──────────────┐    ┌──────────────┐    ┌──────────────┐
│  SQLite DB   │    │  源数据库    │    │  目标数据库  │
│ (系统数据)   │    │ (多源异构)  │    │ (MySQL 8)   │
└──────────────┘    └──────────────┘    └──────────────┘
```

### 2.2 技术栈

#### 前端技术栈
- **框架**: React 18 + TypeScript
- **UI组件库**: Ant Design 5.x
- **状态管理**: React Hooks (Context API)
- **路由管理**: React Router v6
- **HTTP客户端**: Axios
- **代码编辑器**: Monaco Editor
- **图表库**: Recharts
- **构建工具**: Vite

#### 后端技术栈
- **Web框架**: FastAPI 0.109+
- **Python版本**: 3.11+
- **ASGI服务器**: Uvicorn
- **ORM**: SQLAlchemy 2.0+
- **数据库**: SQLite (系统数据存储)
- **任务调度**: APScheduler
- **认证**: JWT (JSON Web Tokens)
- **密码加密**: Passlib + bcrypt
- **向量数据库**: ChromaDB
- **数据库驱动**:
  - PyMySQL (MySQL)
  - cx-Oracle (Oracle)
  - pyodbc (SQL Server)

## 3. 核心模块设计

### 3.1 认证模块

#### 功能描述
- 用户登录/登出
- JWT Token生成与验证
- 验证码生成与校验
- 登录日志记录

#### 实现要点
- 使用JWT进行无状态认证
- Token有效期可配置，默认24小时
- 密码使用bcrypt加密存储
- 验证码存储在内存中（生产环境建议使用Redis）

### 3.2 数据同步模块

#### 功能描述
- 源数据库连接管理（支持多种数据库类型）
- 目标数据库连接管理（MySQL 8）
- 同步任务配置与调度
- 同步状态监控与日志记录

#### 支持的数据库类型
- OceanBase
- StarRocks
- MySQL 5.7/8.0
- Oracle 11g/9c
- SQL Server 2008+

#### 同步策略
- 定时同步（支持每天、每周、每月）
- 全量/增量同步
- 字段级映射配置
- 同步结果通知（系统消息）

### 3.3 知识库模块

#### 功能描述
- 知识文档的增删改查
- 文档向量化存储
- 语义搜索功能
- 支持多种文档格式

#### 技术实现
- 使用ChromaDB作为向量数据库
- 支持中文分词与向量化
- 余弦相似度搜索

### 3.4 科室管理模块

#### 功能描述
- 科室映射关系维护
- 科室排序管理
- 连接外部MySQL数据库（jxks）

#### 外部数据库配置
- IP: 192.168.100.76
- 数据库: jxks
- 表: sys_depts, kspx

## 4. 部署架构

### 4.1 容器化部署

系统支持Docker Compose容器化部署，包含以下服务：
- backend: 后端API服务
- frontend: 前端静态文件服务
- (可选) nginx: 反向代理

## 5. 安全设计

### 5.1 认证安全
- JWT Token认证
- 密码bcrypt加密
- 验证码防暴力破解
- Token过期自动失效

### 5.2 数据安全
- 数据库密码加密存储
- 敏感信息环境变量配置
- SQL注入防护（使用参数化查询）

### 5.3 接口安全
- CORS跨域配置
- 请求限流（可扩展）
- 接口权限验证
