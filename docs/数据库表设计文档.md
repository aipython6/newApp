# 医院数据管理系统 - 数据库表设计文档

## 1. 系统数据库 (SQLite)

### 1.1 用户表 (users)

| 字段名 | 类型 | 长度 | 允许NULL | 说明 |
|--------|------|------|----------|------|
| id | INTEGER | - | 否 | 主键，自增 |
| username | VARCHAR | 50 | 否 | 用户名，唯一索引 |
| password_hash | VARCHAR | 255 | 否 | 密码哈希值 |
| is_active | BOOLEAN | - | 否 | 是否启用，默认true |
| created_at | DATETIME | - | 否 | 创建时间 |

**索引**:
- `idx_users_username`: UNIQUE INDEX on username

### 1.2 登录日志表 (login_logs)

| 字段名 | 类型 | 长度 | 允许NULL | 说明 |
|--------|------|------|----------|------|
| id | INTEGER | - | 否 | 主键，自增 |
| user_id | INTEGER | - | 否 | 用户ID，外键 |
| login_time | DATETIME | - | 否 | 登录时间 |
| ip_address | VARCHAR | 50 | 是 | 登录IP地址 |
| user_agent | VARCHAR | 255 | 是 | 浏览器UA |

**外键**:
- `user_id` → `users.id`

### 1.3 源数据库连接表 (source_db_connections)

| 字段名 | 类型 | 长度 | 允许NULL | 说明 |
|--------|------|------|----------|------|
| id | INTEGER | - | 否 | 主键，自增 |
| name | VARCHAR | 100 | 否 | 连接名称 |
| db_type | VARCHAR | 50 | 否 | 数据库类型 |
| host | VARCHAR | 255 | 否 | 主机地址 |
| port | INTEGER | - | 否 | 端口 |
| database | VARCHAR | 100 | 否 | 数据库名 |
| username | VARCHAR | 100 | 是 | 用户名 |
| password | VARCHAR | 255 | 是 | 密码（加密存储） |
| created_at | DATETIME | - | 否 | 创建时间 |
| updated_at | DATETIME | - | 否 | 更新时间 |

**db_type 可选值**:
- `oceanbase`
- `starrocks`
- `mysql8`
- `mysql5.7`
- `oracle11g`
- `oracle9c`
- `sqlserver2008`

### 1.4 目标数据库连接表 (target_db_connections)

| 字段名 | 类型 | 长度 | 允许NULL | 说明 |
|--------|------|------|----------|------|
| id | INTEGER | - | 否 | 主键，自增 |
| name | VARCHAR | 100 | 否 | 连接名称 |
| host | VARCHAR | 255 | 否 | 主机地址 |
| port | INTEGER | - | 否 | 端口 |
| database | VARCHAR | 100 | 否 | 数据库名 |
| username | VARCHAR | 100 | 是 | 用户名 |
| password | VARCHAR | 255 | 是 | 密码（加密存储） |
| created_at | DATETIME | - | 否 | 创建时间 |
| updated_at | DATETIME | - | 否 | 更新时间 |

### 1.5 同步任务表 (sync_tasks)

| 字段名 | 类型 | 长度 | 允许NULL | 说明 |
|--------|------|------|----------|------|
| id | INTEGER | - | 否 | 主键，自增 |
| task_name | VARCHAR | 100 | 否 | 任务名称 |
| source_db_id | INTEGER | - | 否 | 源数据库ID，外键 |
| target_db_id | INTEGER | - | 否 | 目标数据库ID，外键 |
| source_table | VARCHAR | 100 | 否 | 源表名 |
| target_table | VARCHAR | 100 | 否 | 目标表名 |
| field_mapping | JSON | - | 是 | 字段映射关系 |
| sync_frequency | VARCHAR | 50 | 否 | 同步频率 |
| sync_time | DATETIME | - | 是 | 同步时间点 |
| start_time | DATETIME | - | 否 | 开始时间 |
| last_sync_status | VARCHAR | 20 | 否 | 上次同步状态 |
| last_sync_time | DATETIME | - | 是 | 上次同步时间 |
| is_active | BOOLEAN | - | 否 | 是否启用 |
| created_at | DATETIME | - | 否 | 创建时间 |
| updated_at | DATETIME | - | 否 | 更新时间 |

**sync_frequency 可选值**:
- `daily`: 每天
- `weekly_0` ~ `weekly_6`: 每周日到周六
- `monthly_1`, `monthly_15`: 每月1号或15号

**last_sync_status 可选值**:
- `pending`: 待执行
- `success`: 成功
- `failed`: 失败

**外键**:
- `source_db_id` → `source_db_connections.id`
- `target_db_id` → `target_db_connections.id`

## 4. 初始化数据

### 4.1 默认管理员用户

系统初始化时会创建默认管理员用户：
- 用户名: `admin`
- 密码: `admin123`

**注意**: 生产环境请务必修改默认密码！

### 4.2 数据库初始化SQL

```sql
-- SQLite 数据库会通过 SQLAlchemy ORM 自动创建
-- 以下为参考SQL

-- 创建用户表
CREATE TABLE users (
    id INTEGER PRIMARY KEY AUTOINCREMENT,
    username VARCHAR(50) UNIQUE NOT NULL,
    password_hash VARCHAR(255) NOT NULL,
    is_active BOOLEAN DEFAULT 1,
    created_at DATETIME DEFAULT CURRENT_TIMESTAMP
);

-- 插入默认管理员
INSERT INTO users (username, password_hash, is_active) 
VALUES ('admin', '$2b$12$...', 1);
```

## 5. 数据备份建议

### 5.1 系统数据库 (SQLite)

```bash
# 备份
cp hospital_system.db hospital_system.db.backup.$(date +%Y%m%d)

# 恢复
cp hospital_system.db.backup.20240101 hospital_system.db
```

### 5.2 MySQL 数据库

```bash
# 备份
mysqldump -h 192.168.100.76 -u root -p jxks > jxks_backup_$(date +%Y%m%d).sql

# 恢复
mysql -h 192.168.100.76 -u root -p jxks < jxks_backup_20240101.sql
```
